5주 검증용 샘플 SO 안내

생성 위치
- Assets/Data/Test/WeekValidationSamples

권장 연결 방법
1. WeekFlowController._weekDefinition 에 1주차 asset 연결
2. WeekFlowController._weekDefinitions 에 1주차부터 5주차까지 순서대로 연결

기본 사용 가이드
- 모든 카드는 첫 번째 옵션만 선택해도 핵심 테스트가 보이도록 구성되어 있습니다.
- 각 카드 제목에는 기본 선택 시 무엇이 검증되는지가 적혀 있습니다.
- 각 이벤트 제목과 본문에는 어떤 조건을 확인하는 테스트인지 적혀 있습니다.
- 대화는 `대화 내용 1`, `대화 내용 2` 형식으로 순서가 보이게 작성되어 있습니다.
- 선택 결과 화면은 `결과 내용 1`, `결과 내용 2` 형식으로 표시됩니다.

주차별 핵심 포인트
- 1주차: 시맨틱, 필수 플래그, 선택지 결과, 다음 스텝 이동 확인
- 2주차: 정보 타입 조건과 시맨틱 필터 조건 비교
- 3주차: 주차 시작 효과, 스탯 조건, 완료 후 플래그 연계 확인
- 4주차: 멀티 스텝, 선택지 분기, blocked fallback 대화 확인
- 5주차: 마지막 주차 이벤트 흐름과 엔딩 진입 확인
